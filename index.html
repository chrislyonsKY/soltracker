SolTracker v1.0.0

A 3D Mars rover tracking application that displays all four NASA Mars rovers on a photorealistic globe with sol-by-sol traverse animation.

Data Sources

  • Mars imagery & elevation — Esri Astro / USGS
  • CTX 5m mosaic — Murray Lab, Caltech
  • Rover traverses — NASA MMGIS / PDS
  • Rover photos — NASA Mars Photos API
  • Weather — Curiosity REMS RSS feed
  • 3D models — NASA 3D Resources
  • Geological tiles — NASA Mars Trek WMTS

Built With

ArcGIS Maps SDK for JavaScript 5.0, Calcite Design System, TypeScript, Vite

Credits
